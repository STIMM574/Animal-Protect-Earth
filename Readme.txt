1. 첨부된 APE.txt 와 ape(이미지 폴더)를 다운받습니다.
APE.txt 의 코드를 복사합니다.

2. 파이참 프로그램 에서 새로운 프로젝트 (APE)를 생성하고
최상단의 APE를 우클릭하여 NEW에서 python file을
생성해줍니다. 

3. 복사해둔 APE.txt의 코드를 APE.py(생성시킨 파이썬 파일)에 붙여넣기 해줍니다.

4. ape 이미지 폴더를 APE 프로젝트의 경로에 넣어줍니다.

5. 파이참 프로그램에서 왼쪽 하단의 terminal을 열어서
pip install pygame 이라고 입력합니다. (파이게임 라이브러리 다운로드) 

6. APE.py를 실행(RUN)시켜줍니다. 

7. 게임이 실행됩니다.
